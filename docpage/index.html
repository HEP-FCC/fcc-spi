About

FCCSW is a set of software packages and standards to help different FCC studies work together. Common software helps to avoid duplicated effort and compare results. In addition, the software group provides infrastructure and services such as build systems, testing and continuous integration, code format guidelines, linting and static analysis, release management and software distribution and data persistency. This is possible due to the kind support of the EP-SFT group.

Conceptual Design Report

The summary volumes have been released, with many physics and detector simulations prodced in FCCSW. We collaborate with CERN's Re-usable Analysis working group to make the workflows used in the CDR reproducible and accessible for future studies. Documentation, Tutorials and Getting Started:

Basically all development happens on github.com/HEP-FCC. The best place to check for detailed information on a specific package is usually its github repository page. To get an overview, there are some tutorials that may be helpful. Finally there is a mailing list for any questions and suggestions.
