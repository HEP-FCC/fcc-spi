About FCCSW

FCCSW is the software for the FCC experiments. We aim to support all detector concepts for all accelerator options (electron-electron, electron-hadron and hadron-hadron). Therefore, we try to create software that is independent of detector or experiment specifics.

Conceptual Design Report

The summary volumes have been released, with many physics and detector simulations prodced in FCCSW. We collaborate with CERN's Re-usable Analysis working group to make the workflows used in the CDR reproducible and accessible for future studies. Here is a first example of a full-simulation workflow.

Documentation, Tutorials and Getting Started:

Basically all development happens on github.com/HEP-FCC. The best place to check for detailed information on a specific package is usually its github repository page. To get an overview, there are some tutorials that may be helpful. Finally there is a mailing list for any questions and suggestions.
